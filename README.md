# AI Resume Analyzer

An AI-based web application that analyzes resumes against job descriptions.

## Features
- Upload multiple resumes (PDF/DOCX)
- ATS score calculation
- Keyword matching
- Accept / Reject decision
- Resume ranking system

## Tech Stack
- Python (Flask)
- spaCy (NLP)
- scikit-learn
- HTML, CSS

## How to Run
pip install -r requirements.txt  
python app.py
